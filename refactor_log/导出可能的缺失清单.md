````markdown
## ❌ 不需要导出（通过 builtins 模块访问）

所有 `builtins` 模块中的函数均通过以下路径访问：

```llvm
%print = call ptr @rt_load_global(ptr %module, ptr @.str.print)

%_     = call ptr @rt_call(ptr %print, ptr %args, ptr null)

```cpp
// 将可迭代对象解包为固定长度的 C 数组
PYLANG_EXPORT_SUBSCR("unpack_sequence", "void", "obj,i32,ptr")
void rt_unpack_sequence(py::PyObject *iterable, int32_t count, py::PyObject **out)
{
    auto *iter = rt_unwrap(iterable->iter());
    for (int32_t i = 0; i < count; ++i) {
        out[i] = rt_unwrap(iter->next());
    }
    // 验证迭代器已耗尽
    auto extra = iter->next();
    if (extra.is_ok()) {
        rt_raise(py::value_error("too many values to unpack (expected {})", count));
    }
}
```
````
